# Intel Sustainability Impact Analysis (SQL Project)

📌 Overview
This project analyzes Intel's device repurposing program to evaluate its environmental impact. Using SQL, I explored trends in energy savings and CO₂ reduction across device types, ages, and regions.

🛠 Skills Used
- SQL (Joins, CTEs, Aggregations)
- Data Analysis
- Business Insights & Recommendations

📊 Key Insights
- Over 600,000 devices repurposed in 2024
- ~1.68 million kWh saved annually
- Equivalent to powering ~160 homes
- ~846 tons of CO₂ saved (≈ 184 cars)

🌍 Key Findings
- Older devices yield higher energy savings per unit
- Newer devices are repurposed more frequently
- Asia delivers the highest CO₂ reduction impact

💡 Recommendations
- Prioritize repurposing older devices
- Focus efforts in carbon-intensive regions
- Extend lifecycle of newer devices

📁 Project Structure
- `queries/` → SQL scripts
- `insights/` → analysis & findings

🚀 How to Use
Run the SQL queries in order using any SQL environment with the provided dataset.
